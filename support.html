What to Include

To help resolve issues quickly, include the following in your message:

  • A brief description of the issue or question
  • The device you are using (e.g. iPhone 14, Samsung Galaxy S22)
  • The operating system version (e.g. iOS 18.1, Android 14)
  • Steps to reproduce the issue, if applicable
