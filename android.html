How to Install

Because this app is distributed directly rather than through the Google Play Store, Android will ask you to allow installation from an external source. Follow these steps:

  1. 1
    Download the APK using the button above. Your browser may warn you that the file could be harmful — tap Download anyway. The file is safe.
  2. 2
    Open the downloaded file. You can find it in your browser's downloads or in the Files app under Downloads.
  3. 3
    Allow installation from this source if prompted. Android will show a dialog asking permission — tap Settings, enable Allow from this source, then go back and tap Install.
  4. 4
    Tap Install on the confirmation screen. The app will install in a few seconds and appear on your home screen.

Is This Safe?

Yes. AlphaProject is built specifically for Campus Outreach Alpha Project participants. All personal data — journal entries, goals, favorites — is stored locally on your device and never leaves it.

The app makes two types of network requests: fetching announcements posted by staff, and if you grant notification permission, registering a randomly generated device token with Firebase Cloud Messaging (Google) to deliver push notifications. That token is not linked to your name, email, or any personal information and is used only to route notifications to your device.

No personal information is collected, no ads are shown, and no data is shared with third parties beyond Firebase for notification delivery.

If you have any concerns or run into issues during installation, reach out via the Support page.

Updating the App

When a new version of AlphaProject is available, a new APK will be posted here. To update, simply download and install the new APK — it will replace the existing version without deleting your data.
